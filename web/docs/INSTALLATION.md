# 安装指南

## 首次配置（做一次即可）

### 前置检查

| 组件 | 版本要求 | 检查方式 |
|---|---|---|
| Windows | 10/11 x64 | `winver` |
| 微信 PC 版 | 3.9+ | 微信 → 设置 → 关于 |
| Python | 3.9+ | `python --version` |
| ffmpeg | 已内置 | 项目里的 `tools/video/ffmpeg/ffmpeg.exe` |

> Python 只用标准库，无需 `pip install` 任何依赖。

---

### 第 1 步：启动下载器

1. 进入项目目录 `d:\AI_bian_cheng\trae\tools\wx_video-channel\`
2. **双击 `start.vbs`**（不要直接双击 `蝴蝶号下载助手.exe`，会缺少提权）
3. UAC 弹窗 → 点"**是**"（**必须**以管理员启动，否则微信里不会出现下载按钮）
4. 首次启动会自动安装 SunnyNet 根证书 → 全部选**是**
5. 3~5 秒后浏览器会自动打开 `http://127.0.0.1:2025/console`

看到控制台首页说明后端已经就绪。

---

### 第 2 步：让微信生效

驱动注入需要在**微信启动之前**完成，所以：

1. **完全退出微信**（右下角托盘图标 → 右键 → 退出，不是关闭窗口）
2. 重新打开微信 → 登录
3. 侧边栏点"**蝴蝶号**"
4. 播放任意视频 → 视频下方按钮栏应出现 **下载图标（↓）**

如果没出现下载按钮，去看 [故障排除](TROUBLESHOOTING.md) 的第 1 条。

---

### 第 3 步：证书未装成功的手动补装（少见）

如果第 1 步显示"证书安装失败"：

1. 打开 `downloads\SunnyRoot.cer`
2. 双击文件 → 点 **安装证书**
3. 选 **本地计算机** → **将所有证书放入以下存储** → 浏览 → **受信任的根证书颁发机构** → 确定
4. 完成后重启微信

---

## 首次使用流程

```
蝴蝶号看到好视频
   ↓ ① 双击 start.vbs（如果已在跑，跳过）
   ↓ ② 在蝴蝶号播放页点"下载"按钮 → mp4 落盘
   ↓ ③ 下载器自动抽 mp3 + 转 md（v1.1 无需手动）
   ↓ ④ 在你当前用的 AI 助手对话里发"清洗"，触发 SKILL 生成清洗版稿子

最终每个视频目录只留 3 个文件：mp4 + mp3 + _清洗版.md
```

---

## 关闭下载器

**不需要专门关。** 本工具通过驱动只注入 `WeChatAppEx.exe`，**从不修改系统代理**，直接关机、崩溃、任务管理器强杀都不会残留任何东西、不会影响其他软件上网。

如果想主动关：任务管理器结束 `蝴蝶号下载助手.exe`（提权进程需要以管理员打开任务管理器）即可。

看门狗 `guard.vbs` 会在微信完全退出满 5 分钟后自动帮你关掉下载器。

---

## 升级到新版本

1. 停止旧版本（任务管理器结束 `蝴蝶号下载助手.exe`）
2. 替换 `蝴蝶号下载助手.exe` 文件
3. 双击 `start.vbs` 启动新版

配置文件 `config.yaml` 和 `downloads/` 会保留，不会被覆盖。
