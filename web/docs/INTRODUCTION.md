# 项目介绍

## 这是什么

**蝴蝶号下载助手（本地二开版）** — 基于 Go + SunnyNet 内核网络驱动的本地代理工具，用于把蝴蝶号的加密视频流拦截下来解密并保存为 mp4，随后自动抽音频、转文字，最终配合 AI 清洗出可读的逐字稿。

一句话说明：**在蝴蝶号里点一下下载，你会得到 视频 + 音频 + 可读逐字稿 三个文件。**

> ⚠️ **适用范围**：仅支持微信 App 侧边栏"蝴蝶号"入口的视频。
> 公众号文章视频、朋友圈视频、微信小程序视频**不支持**。

---

## 核心能力

- **一键下载**：蝴蝶号播放页会自动出现下载按钮，点一下视频就落到 `downloads/{作者名}/` 目录
- **🎙️ 自动转录（v1.1 二开新增）**：视频下载完成后，2 个后台 worker 自动完成"抽 mp3 → 调百度短语音识别 API → 生成 md 逐字稿"，全程无需手动
- **🧹 自动洗稿（v1.2）**：Go 后处理流水线下载完成后自动调用 clean.py（需在 config.yaml 配置 llm_api_key），通过 OpenAI 兼容 API 清洗逐字稿，生成结构化可读的 `_清洗版.md`
- **视频信息采集**：自动采集标题、作者、时长、大小、点赞/评论/转发数
- **文件组织**：按作者自动分类保存
- **本地控制台**：`http://127.0.0.1:2025/console` 查看下载记录、统计、配置

---

## 工作原理

1. **驱动注入**：程序启动后通过 SunnyNet 内核驱动**只注入蝴蝶号渲染进程 `WeChatAppEx.exe`**，从不修改系统全局代理，不影响其他软件上网、不影响 Clash 等代理工具
2. **流量拦截**：拦截蝴蝶号 API 流量，采集视频元信息 + 拿到加密流地址
3. **解密下载**：本地服务下载并实时解密视频流，写入本地文件系统
4. **自动后处理（v1.1）**：mp4 落盘触发 PostProcessService，先调 `extract_audio.py --file <mp4>` 生成 mp3，再调 `transcribe.py --file <mp3>` 生成 md
5. **自动洗稿（v1.2）**：Go 后处理自动调用 clean.py（配置 llm_api_key）清洗逐字稿 → `_清洗版.md`
6. **记录管理**：所有下载记录写入 SQLite 数据库 + CSV 文件双备份

## 技术架构

- **后端**：Go 1.23，Gin 风格 HTTP 服务 + SunnyNet 代理内核
- **代理**：内核网络驱动（NF/nfapi），仅注入指定进程
- **前端**：原生 JavaScript 注入脚本 + 本地 Web 控制台
- **存储**：本地文件系统（视频/音频/逐字稿）+ SQLite（元信息）
- **转录**：Python + ffmpeg + 百度短语音识别 API（免费额度 5 万次/日）

---

## 系统要求

- **操作系统**：Windows 10 / 11 x64
- **微信 PC 版**：3.9+
- **Python**：3.9+（`extract_audio.py` / `transcribe.py` 用到）
- **ffmpeg**：已内置在 `tools/video/ffmpeg/`
- **权限**：需管理员启动（驱动注入硬性要求，由 `start.vbs` 自动处理 UAC）
