## 🚀 快速开始

1. 双击 `start.vbs` 启动下载器（UAC 点是）
2. 打开微信 → 蝴蝶号 → 播放视频 → 点下载图标
3. **下载器自动帮你抽 mp3 + 转 md 逐字稿**（v1.1 新特性，无需手动）
4. 在你正在用的 AI 助手对话里发"清洗"，触发 SKILL 生成最终清洗版稿子

## 📁 输出结构

```
downloads/
└── {作者名}/
    ├── {视频标题}.mp4       ← 原始视频
    ├── {视频标题}.mp3       ← 自动抽取的音频（v1.1）
    └── {视频标题}.md        ← 百度 API 转出的原始逐字稿（v1.1）
```
